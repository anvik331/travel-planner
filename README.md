# Travel Planner

Travel Planner is a web app for planning trips with friends. User can add places they want to visit, organize them by priority. The places will be on the map so the user can know the best order to visit each spots. This app will be in both Mandarin and English, so both Mandarin and English user can use it.

The goal is making the traveling plan easier between friends to combine a shared list.

# Project Idea

When planning a trip, people usually saved places in different app. And use maps to search how to get to each spots. This projects is for we can do all these features in one app.

User will be able to
 - Add travel spots
 - View all the spots on the list and map
 - Priortize where they want to go
 - Categorize each spots
 - Assign spots to different dates
 - Check transportation time between spots

This project will start without a login system. The first version will focus on the core planning structure and user interface.
